# Changelog

Notable changes to Kloppy and its launch site are recorded here.

## Unreleased — launch website

### Added

- A downloads-first homepage with centralized, checksum-verified v0.1.0
  artifacts for every platform the project actually publishes.
- A client-side Certificate of Forgiveness generator with download, clipboard,
  and explicit share actions.
- Complete canonical, Open Graph, and Twitter Card metadata, including a branded
  1200 × 630 social preview and favicon.
- Launch notes, pull request notes, a deployment checklist, and a rollback
  procedure.

### Changed

- Qualified the historical 69-commit launch claim so it cannot be mistaken for
  the repository's permanent total.
- Moved the free download flow ahead of the lore and optional $4.20 contribution.
- Updated the white paper and runbooks to describe the shipped local model flow,
  published packages, and voluntary contribution accurately.

## v0.1.0 — 2026-07-12

- Shipped the local-first Electron desktop application with notes, reminders,
  tray behavior, opt-in folder commentary, and local llamafile chat.
- Published unsigned Windows x64, macOS universal, Linux x86_64 AppImage, and
  Linux amd64 Debian artifacts with SHA-256 checksum files.
- Required no account, telemetry, cloud subscription, or payment.

### The 69-commit history

Kloppy v0.1.0 was originally completed in exactly 69 commits. That count refers
to the original application-and-launch state on `main`, which ended at commit
`aa91450`. The `v0.1.0` tag points to the earlier desktop release commit
`a5fdf2a`; the remaining original-launch commits connected the release downloads
and contribution flow.

Later website commits improved launch messaging and distribution. The historical
69-commit claim refers to the original application release, not the repository's
permanent total.
